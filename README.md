## Одностраничный сайт о методах и принципах обучения - "Путешествие по России"

В проекте рассказывается о разных интересных местах России.

#### Технологии:
* HTML5
* CCS3
* Методология БЭМ (наименования классов и файловая структура)
* Flexbox
* Анимации
* Трансформации
* Позиционирование
* Псевдоклассы
